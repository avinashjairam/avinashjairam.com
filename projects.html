Merge My Videos

A web application

This website allows a user to upload a number of videos, rearrange them in a particular order, and join them together. The user is then able to download a single video. It is written using the LAMP stack, HTML,CSS, and JavaScript. Users can currently upload and merge .mp4,.avi,.flv,.mov, and .wmv video formats.
